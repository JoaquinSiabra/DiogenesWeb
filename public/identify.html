DiogenesWeb

DiogenesWeb is an web application for reading XML files of Latin and Ancient Greek. These can be open-access files or can be files held in your private cloud storage.

Before starting to use DiogenesWeb, please indicate which category of user you belong to. This information helps ensure the continued funding of the Diogenes project.

Privacy Notice

DiogenesWeb does not use cookies or store any user information whatsoever on its server; all user information is kept in your browser's local storage. Each time you access DiogenesWeb, the server logs the category of user, nothing more.

When you connect to the website of your cloud storage provider, you will have to enter your password. DiogenesWeb never sees that password; it goes directly to your cloud provider. Your provider sends back a time-limited token that permits access only to the DiogenesWeb folder within your private storage. That token is stored in your browser.
